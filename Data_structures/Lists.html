access single element: list_var[index]
access range of elements: list_var[start:end]
list_var: name of the variable that contains the list
elements: the contents that are being stored in the list
start : index of the first element in the range (inclusive)
end : Last element in the range (exclusive)
Line 2: List creation
creates a list, which can store a series of elements into a single variable

Line 10: Access a range of elements where the start anc end of the range are indicated
Line 12: Access a range of elements where the start is at default and end is indicated
Line 14: Access a range of elements where the start is indicated and the end is end of list

1) A list can contain a mix of datatypes 2) A list can be editted 3) A blank list is created by leaving the section between the brackets blank
