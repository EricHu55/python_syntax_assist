multiple user input shorthand method (unpacking)

code breakdown
user input unpacking: var1,var2,varn = map(datatype,input("message: ").split("delimiter"))
var1,var2,varn : These are the variables that are suppose to store the inputs
map : This ensures that the following function is applied to each value or variable in this case
datatype: this converts the input to the desired datatype as input default datatype is string
input : Allows the user to enter a value during the code execution
.split() : This method divides a string based on the entered delimiter. The default delimiter is by whitespace
delimiter : This determines where the code will split the string. If no value is entered .split will divide the string by whitespace

Line 7: multiple user input shorthand
This allows the user to enter inputs for multiple variables at once
Rules:
1) The datatype for the inputs all have to be the same
2) The number of inputs must match the number of variables
